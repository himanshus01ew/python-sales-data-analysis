# 🛒 E-Commerce Sales Capstone Project

A complete **Data Science Capstone Project** for students using Python, NumPy, Pandas, Matplotlib, and Seaborn on a realistic Indian e-commerce dataset with **10,000 orders**.

## 📁 Project Files

| File | Description |
|------|-------------|
| `ecommerce_sales.csv` | Dataset — 10,000 rows × 17 columns |
| `capstone_notebook.ipynb` | Full analysis notebook (10 modules) |
| `PROJECT_GUIDELINES.docx` | Step-by-step project guide & rubric |
| `README.md` | This file |

## ⚙️ Setup

```bash
pip install pandas numpy matplotlib seaborn jupyter faker
jupyter notebook capstone_notebook.ipynb
```

## 📚 Modules Covered

| # | Module | Key Skills |
|---|--------|-----------|
| 1 | Setup & Loading | File I/O, pandas basics |
| 2 | Data Inspection | dtypes, info, describe |
| 3 | Data Cleaning | Missing values, type casting |
| 4 | Python Core & OOP | Lambda, list comp, classes |
| 5 | NumPy | Vectorised ops, statistics |
| 6 | Pandas EDA | GroupBy, pivot, RFM, resample |
| 7 | Matplotlib | Line, bar, pie, scatter, subplots |
| 8 | Seaborn | Heatmap, boxplot, violin, pairplot |
| 9 | Business Insights | KPIs, growth rates, recommendations |
| 10 | GitHub Workflow | Git init, commit, push |

## 📊 Dataset Columns

`order_id`, `customer_id`, `order_date`, `order_time`, `customer_age`, `customer_gender`, `customer_city`, `city_tier`, `category`, `product_name`, `unit_price`, `quantity`, `discount_pct`, `total_price`, `payment_method`, `order_status`, `rating`
